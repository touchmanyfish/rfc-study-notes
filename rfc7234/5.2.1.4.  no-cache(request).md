# 5.2.1.4.  no-cache
[rfc 7234 5.2.1.4.  no-cache](https://www.rfc-editor.org/rfc/rfc7234#section-5.2.2.7)

## 遇到no-cache时使用哪种方式进行验证？
```text
   The "no-cache" request directive indicates that a cache MUST NOT use
   a stored response to satisfy the request without successful
   validation on the origin server.
```
那么问题来了，验证方式有多种：
```text
If-Match
If-Unmodified-Since
If-None-Mathc
If-Modified-Since
```
